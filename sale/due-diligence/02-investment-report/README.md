# 02 — Investment Report

## Purpose

Deeper analysis of the asset as an investment thesis: what it cost to build, what it costs to operate, what the plausible unit economics look like at various scale points, and how the asking price relates to replacement cost. Investor-oriented, factual, no ARR / MRR claims (pre-revenue is explicit throughout).

## Files that will be added

- `investment-report.docx` — polished Word document (regenerated via `sale/_build_docx.py`)
- `investment-report.md` — source Markdown
- `build-cost-analysis.md` — hours × market rate × developer seniority × complexity, with sensitivity ranges
- `cost-of-operation-model.xlsx` — Render / Brevo / Cloudflare / AI provider cost projections at 10, 100 and 1000 practices
- `valuation-framework.md` — comparable asset benchmarks, replacement-cost floor, opportunity-value ceiling
- `pricing-strategy.md` — reference to the current `Basic $29 / Pro $59 / Unlimited $99` tier structure, with alternative tier scenarios modelled

## NDA classification

**Share after NDA.** The report contains subjective valuation judgements, cost projections, and comparable-asset references that are useful to serious buyers but should not be public. Provided upon receipt of a signed mutual NDA.

## Reader notes

- Numbers are illustrative planning tools, not forecasts. Buyer builds their own model.
- The build-cost analysis is grounded in actual solo-founder hours over the six-month build period — hours logged in the seller's private time-tracking, verifiable on request.
- The valuation framework references pre-revenue SaaS asset sales on Flippa and Acquire.com from 2024–2026 as comparable set.
