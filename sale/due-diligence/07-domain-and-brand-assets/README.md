# 07 — Domain & Brand Assets

## Purpose

Documents the ownership of the `praxisonline24.com` domain and the transferable brand kit. Answers *"Do I get real IP or just an idea?"*

## Files that will be added

- `domain-registration-proof.pdf` — whois excerpt or registrar receipt confirming ownership
- `domain-transfer-instructions.md` — step-by-step transfer via unlock + auth code, typical duration
- `trademark-search-report.md` — global trademark search results ("PraxisOnline24" is not registered — buyer may register in target market)
- `brand-guidelines.md` — colour palette, typography, logo usage rules
- `logo-variants/` — SVG + PNG in various sizes (mirrors `sale/assets/branding/`)
- `favicon-set.zip` — favicon in all required sizes (16, 32, 48, 96, 192, 512)
- `wordmark.svg` / `wordmark.png` — brand wordmark

## NDA classification

**Public** for the brand kit previews (logo, colour palette, marketing screenshots). **Share after NDA** for the transfer specifics (registrar account context, auth codes will only be shared during escrow — see section 09).
