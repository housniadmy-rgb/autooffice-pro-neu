# 01 — Executive Summary

## Purpose

A concise, buyer-facing overview of the PraxisOnline24 asset. Intended to be read in under three minutes and to answer the question *"What is this and why should I care?"* before the buyer commits time to deeper materials.

## Files that will be added

- `executive-summary.pdf` — polished single-page PDF (regenerated from source Markdown via `sale/_build_pdf.py`)
- `executive-summary.md` — source Markdown for the PDF
- `one-pager-hero-image.png` — cover image (matches Flippa listing cover)
- `key-metrics-snapshot.md` — headline figures (build effort, dependencies count, languages supported, deployment stack)

## NDA classification

**Public.** Everything in this folder can be shared with any prospective buyer without a signed NDA. Content is deliberately marketing-oriented and does not disclose sensitive implementation detail, credentials or legal terms.

## Reader notes

- The Executive Summary aligns with the Flippa listing description at `sale/flippa/listing-draft.md`. If the two ever diverge, the Flippa listing text is authoritative.
- Do not upload the PDF to the Flippa "documents" slot until DD Rev.1 is finalised — a placeholder is fine during the initial listing phase.
