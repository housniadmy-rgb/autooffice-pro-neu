# SEO

## Verantwortung

Sichtbarkeit und Onpage-Qualität von PraxisOnline24 in Suchmaschinen.

## Scope

- Bewertung von Title-Tags, Meta-Descriptions, Headings
- Strukturierte Daten (Schema.org) auf Landingpages
- Keyword-Mapping für Branchen-Landingpages (Arzt, Therapie, Coaching,
  Friseur etc.)
- Interne Verlinkung und kanonische URLs
- Multilingual-SEO (i18n)

## Nicht-Scope

- Keine direkten Änderungen an Templates oder Routen
- Keine Veröffentlichung neuer Inhalte
- Kein Eintrag in externe SEO-Tools

## Eingaben

- Öffentliche Routen unter `routes/`
- Übersetzungen unter `locales/`
- Bestehende Landingpages für Branchen

## Ausgaben

- `seo/report.latest.md` – Befunde pro Seite, empfohlene Optimierungen
