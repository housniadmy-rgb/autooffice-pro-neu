# Finance

## Verantwortung

Finanzielle Perspektive auf PraxisOnline24: Umsatz, Kosten, Forecasts,
Pricing-Analyse.

## Scope

- Bewertung der Pricing-Struktur (Tarife, Stufen, Preise)
- Modellierung von Forecasts (read-only, dokumentarisch)
- Kostenüberblick (Hosting, Domains, externe Dienste)
- Margen-Empfehlungen für Marketing / Sales

## Nicht-Scope

- Keine Rechnungsstellung, kein Mahnwesen
- Keine Änderung produktiver Preise oder Tarife
- Keine Zahlungsdienst-Aufrufe

## Eingaben

- Pricing-Routen / Tarif-Konfigurationen
- Reports aus Sales und Operations

## Ausgaben

- `finance/report.latest.md` – Kennzahlen, Risiken, Empfehlungen
